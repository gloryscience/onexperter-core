# OnExpert v3.0 Changelog

## Major Changes

1. Elevated digital twins from an optional visualization topic to a core professional capability.
2. Added DT0–DT5 digital-twin maturity and AO0–AO5 autonomy maturity.
3. Added TwinGraph, TwinCopilot, Temporal Twin, Self-Calibrating Twin and Counterfactual Twin.
4. Added SafeAction Sandbox and simulation-guarded production actions.
5. Added explicit real/simulated/inferred/stale/unknown data labels.
6. Added engineering-grade modeling workflow and L2/L3/L4 acceptance criteria.
7. Added unified multi-vendor semantic model.
8. Added AI Agent collaboration and Twin-to-Knowledge learning loop.
9. Preserved Expert First, Product Second and strengthened Oneasy non-overclaim rules.
10. Expanded active Knowledge from 12 to 14 files while remaining below the 15-file limit.
